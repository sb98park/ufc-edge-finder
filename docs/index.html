🔍 How the model works
▶

Base rating: blends Elo (built from recorded fight history, when it exists) with a stats-based rating from career win rate, finish rate, experience, and reach — so a fighter with no tracked fight history isn't just defaulted to a flat, meaningless 50/50.

Style matchup adjustment ("styles make fights"): takedown accuracy vs. the opponent's takedown defense (wrestling advantage), striking accuracy differential, and durability (how often each fighter has actually been finished before, and by what method) all shift the base rating gap before it's converted to a win probability.

Method-of-victory props: blend a fighter's own career finish tendency (65% weight) with how often this specific opponent has been finished that way before (35% weight) — a big KO rate matters less against someone who's never been knocked out.

Total rounds: a simplified proxy using both fighters' combined finish rate.

Honest limitations: striking/takedown/durability stats are sample data for now (not yet a live feed from UFCStats or similar) — see the README for how to refresh them for real accuracy. This is a systematic way to weigh public stats, not a trained predictive model, and it can't see recent form, injuries, or camp changes.
